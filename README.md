# Оно тебе надо

Оно тебе надо - маленький проект, об аукционе.

## Description

В проекте присутствуют:
- Шапка
- Основная часть, которая состоит из 3 секций
- Подвал

### Технологии, которые были использованы
- Flexbox
- Grid
- Positioning
- Semantic tags like 'address', 'article', 'nav', 'section' etc.
